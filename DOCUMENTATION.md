# Skillpack Documentation

## Overview

Skillpack is a Claude Code plugin marketplace hosted at `github.com/p3ob7o/skillpack`. Users add it with `/plugin marketplace add p3ob7o/skillpack` and install plugins with `/plugin install <name>@skillpack`.

## Architecture

```
skillpack/
  .claude-plugin/marketplace.json   # Marketplace catalog
  .gitignore                        # Excludes voice/SKILL.md, .DS_Store
  LICENSE                           # GPL v2
  README.md                         # Top-level marketplace README
  plugins/
    writewell/                      # First plugin
      .claude-plugin/plugin.json    # Plugin manifest
      README.md                     # Plugin README with install/usage
      skills/
        writewell/
          SKILL.md                  # Main skill — writing pipeline
          onboard/SKILL.md          # Onboarding — generates voice profile
          voice/
            SKILL.md                # Personal voice (generated, gitignored)
            README.md               # Explains the voice directory
            .gitkeep                # Keeps directory in git
```

## Plugins

### writewell (v1.4.0)

Five-part writing pipeline:
1. **AI pattern removal** — strips LLM tells (significance inflation, filler, hedging, chatbot artifacts)
2. **Personal voice** — applies user's writing style from `voice/SKILL.md`
3. **Language discipline** — kills dead metaphors, deflates padded verbs, prefers short words (Orwell)
4. **Clutter removal** — three revision passes: cut, clarify, reader-test (Zinsser)
5. **Quality check** — natural voice, clear purpose, brevity (Ogilvy)

**Onboarding flow:** On first run, a bash check detects missing `voice/SKILL.md` and reads `onboard/SKILL.md` directly. The onboard routine presents three options via AskUserQuestion:
- Post URLs — user pastes individual article links
- Blog homepage — fetches N most recent posts in reverse chronological order
- Skip — writes a placeholder so onboarding doesn't repeat

**Voice profile:** Generated by analyzing writing samples across 11 dimensions (sentence structure, tone, vocabulary, openings, closings, formatting, rhetorical devices, point of view, emotional register, distinctive touches). Stored as `voice/SKILL.md`, gitignored since it's personal.
